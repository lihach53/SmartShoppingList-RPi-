# Умный список покупок

<details>
<summary><strong>О проекте</strong></summary>
   
---

Умный список покупок - это Android-приложение с клиент-серверной архитектурой, позволяющее синхронизировать списки покупок между разными устройствами в реальном времени.

---

## Основной функционал

### Клиентская часть (Android приложение)
- **Просмотр списка продуктов** - отображение всех товаров для покупки
- **Добавление новых продуктов** - интуитивный интерфейс для добавления позиций
- **Отметка купленных продуктов** - возможность отмечать приобретенные товары
- **Удаление продуктов** - удаление ненужных позиций из списка
- **Автоматическая синхронизация** - своевременное обновление данных между устройствами

###  Серверная часть (Raspberry Pi)
- **Хранение данных** - централизованное хранение списка продуктов
- **REST API** - обработка запросов от всех подключенных устройств
- **Real-time синхронизация** - обеспечение актуальности данных на всех клиентах
- **Управление подключениями** - обработка множественных одновременных подключений

---

## Интерфейс приложения

### Основной экран:
1. **Главный экран списка покупок**
   - Отображение всех товаров
   - Статус покупки (куплено/не куплено)
   - Кнопки действий
   - Поле ввода названия товара
   - Добавление заметки к товару (опционально)
